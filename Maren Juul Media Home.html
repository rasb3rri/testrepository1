    Open Source Tools for Design and Copywriting

    All of the software I enjoy using is Open Source. This means the software was created under copyright licenses which allows a user to use, view and change the code of the software as they wish.

    For this site, I decided to write my own HTML and CSS directly, just to give you an idea of what I can do for you. I've created this website from scratch in an open source program known as Notepad++. Unfortunately this program can be dangerous to download due to hackers adding their own malware to the download and creating chaos for the user. The malware was known as StrongPity. Thankfully, this virus can be avoided by downloading the Notepad++ program from a reputable source. For this reason, I will only recommend you visit this link to their official website: https://notepad-plus-plus.org/downloads/.

    Second, you should know, I've created all the art and design elements on this website with a few Open Source programs, InkScape and Krita and GIMP. These three replace Adobe programs and most people doing reviews out there agree, these are very valuble, powerful programs that are easy to use. I higly recommend Krita for anyone that likes to paint, it works very well with a drawing tablet and the brushes are very realistic and you can customize practically any parameter.

    My favorite program for logo design is Inkscape, its a lot like Adobe Illustrator. You can use it to create vector graphics, trace png images and it was really easy to learn the basics by using tutorials on YouTube, I would highly recommend these for beginners who like apps like Canva and want more functionality. The only thing that I feel is a negative, is that it tends to crash if you're working with scanned images and there are too many nodes, so I often use GIMP to clean up photos before importing them to InkScape to redraw them or trace. If you're looking to give these programs a try, then please check out these for safe websites to download them from.
